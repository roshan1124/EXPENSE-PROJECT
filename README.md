This project is a simplified expense sharing backend system, inspired by applications like Splitwise.
It allows users to create groups, add shared expenses, and track balances showing who owes whom.

The system is intentionally kept simple and uses in-memory storage to focus on logic, design, and clarity, as required for the assessment.

Features
Create users
Create groups with multiple users
Add shared expenses
Track balances (who owes whom)
Simplified balance calculation
Supports multiple split types:
Equal split
Exact amount split
Percentage split

Tech Stack
Node.js
Express.js
JavaScript
In-memory data structures (no database)
